Qwen Coder 모델별 대략 사양

모델 권장 메모리 그래픽카드 비고
0.5B RAM 8GB 없어도 가능 가벼운 테스트용
1.5B RAM 8GB 없어도 가능 자동완성, 가벼운 실습용
3B RAM 8GB~16GB 없어도 가능 일반 노트북 실습 가능
7B RAM 16GB 권장 NVIDIA VRAM 8GB 이상이면 좋음 이번 실습 추천
14B RAM 32GB 권장 NVIDIA VRAM 12GB~16GB 권장 속도 부담 있음
32B RAM 32GB~64GB 권장 NVIDIA VRAM 24GB 이상 권장 개인 PC 실습에는 무거움

NVIDIA 그래픽카드가 있으면 GPU 가속을 쓰기 쉽습니다. NVIDIA가 아니어도 Windows에 Ollama 설치는 가능하지만, GPU 가속이 안 잡히면 CPU 실행에 가까워져 많이 느릴 수 있습니다. WSL/Linux 설치는 Linux 개발 환경이나 CUDA 구성이 필요할 때 선택하는 방식이지 필수는 아닙니다.

실습 순서

아래 커맨드는 전부 오른쪽 위 복사 버튼으로 복사할 수 있습니다.

  1. Ollama 설치

    Windows: Ollama Windows 다운로드 에서 설치 파일을 받아 실행합니다.

    Linux 환경에서는 WSL이나 Docker를 이용해 설치할 수도 있습니다. 이 교육에서는 Windows 설치 기준으로 진행합니다.

  2. PowerShell 또는 CMD 열기

    설치가 끝나면 PowerShell 또는 CMD를 열어 명령어를 실행합니다.

    • 방법 1: `Win` 키를 누르고 `PowerShell` 또는 `cmd`를 검색해서 실행
    • 방법 2: `Win + X`를 누른 뒤 `터미널` 또는 `PowerShell` 선택
    • 방법 3: `Win + R`을 누르고 `cmd` 입력 후 Enter

    일반 PowerShell 또는 CMD로 실행하면 됩니다. 관리자 권한은 보통 필요하지 않지만, 회사 PC 보안 정책이나 설치 권한 문제로 실패하면 관리자 권한으로 다시 시도합니다.

  3. Ollama 설치 확인

    ollama --version

  4. 모델 다운로드

    Linux 환경: ollama serve & 커맨드 먼저 실행하여 서버 가동. &: 백그라운드 실행

    참고: 뒤의 :7b 부분을 :3b, :1.5b처럼 바꾸면 더 작은 모델을 사용할 수 있습니다. 장비 사양이 충분하면 더 큰 모델 태그로 변경할 수도 있습니다. 

    ollama pull qwen2.5-coder:7b

  5. 모델 목록 확인

    ollama list

  6. 모델 실행

    ollama run qwen2.5-coder:7b

  7. 실행 중인 모델 확인

    ollama ps

VS Code에서 Continue 추가

아래 영상을 보고 VS Code에 Continue 확장을 추가한 뒤 다음 설정을 진행합니다.

Continue YAML 설정

Continue는 `config.yaml` 파일에 Ollama 모델 정보를 적어서 연결합니다.

설정 파일 위치

C:\Users\사용자명\.continue\configs\config.yaml

작성 예시

name: Local Qwen
version: 0.0.1
schema: v1

models:
  - name: Qwen2.5 Coder 7B
    provider: ollama
    model: qwen2.5-coder:7b
    apiBase: http://localhost:11434
    roles:
      - chat
      - edit
      - apply
    defaultCompletionOptions:
      contextLength: 4096
      temperature: 0.2
  • `model` 값은 `ollama list`에 보이는 모델명과 정확히 같아야 합니다.
  • 교육에서는 `qwen2.5-coder:7b`처럼 태그를 명확히 씁니다.
  • 메모리 부족이 나면 `contextLength`를 낮추거나 더 작은 모델을 사용합니다.

간단 실습

C# 코드 설명, 리팩터링, 메일 작성까지 진행합니다.

실습 1: C# 코드 설명

public List<string> FindDuplicates(List<string> items)
{
    var seen = new HashSet<string>();
    var duplicates = new HashSet<string>();

    foreach (var item in items)
    {
        if (seen.Contains(item))
        {
            duplicates.Add(item);
        }
        else
        {
            seen.Add(item);
        }
    }

    return duplicates.ToList();
}
이 C# 함수가 어떤 일을 하는지 설명하고, 시간 복잡도와 공간 복잡도를 알려줘.

실습 2: C# 코드 리팩터링

public string GetUserStatus(User user)
{
    if (user != null)
    {
        if (user.IsDeleted == false)
        {
            if (user.IsActive == true)
            {
                if (user.LastLoginDate != null)
                {
                    var days = (DateTime.Now - user.LastLoginDate.Value).Days;
                    if (days < 30)
                    {
                        return "Active";
                    }
                    else
                    {
                        return "Dormant";
                    }
                }
                else
                {
                    return "NeverLoggedIn";
                }
            }
            else
            {
                return "Inactive";
            }
        }
        else
        {
            return "Deleted";
        }
    }
    else
    {
        return "Unknown";
    }
}
이 C# 코드를 더 읽기 좋게 리팩터링해줘.
중첩 if를 줄이고, 상태값이 어떻게 결정되는지도 간단히 설명해줘.

실습 3: 메일 작성

내부 API 장애로 배치 작업이 실패했다.
원인은 인증 토큰 만료였고, 토큰 갱신 후 재처리는 완료됐다.
영향 범위는 전일 23:00부터 금일 01:00까지 생성된 일부 리포트다.
위 상황을 바탕으로 운영팀과 개발팀에 보낼 장애 공유 메일 초안을 작성해줘.
원인, 영향 범위, 조치 내용, 재발 방지 계획 순서로 정리해줘.

활용 팁

💡 로컬 AI 200% 활용하는 꿀팁 3가지

로컬 모델은 상황에 맞게 가볍게 쓰는 것이 중요합니다. 느려지면 리셋하고, 작업 난이도에 따라 모델을 바꿔 쓰세요.

1

답변이 갑자기 느려지거나 헛소리하면 새 세션

코딩하다가 AI가 첫 대답을 너무 뜸 들이거나 문맥이 꼬였다고 느껴지면, 고민하지 말고 새 세션으로 기억을 리셋합니다. Continue에서 새 세션 단축키를 Ctrl + Alt + N처럼 지정해두면 빠르게 다시 시작할 수 있습니다.

2

평소에는 3B, 정교한 코딩은 7B

Qwen 2.5 Coder 3B는 가볍고 빨라서 간단한 함수 작성, 주석 달기, 정규식 만들기 같은 작업에 적합합니다. 이번 실습은 7B로 진행하지만, 이후 빠른 작업용 모델이 필요하면 3B를 추가로 받아 사용할 수 있습니다.

3

코드 자동 완성은 Tab으로 받기

Continue에서 자동완성 모델을 지정해두면 소스코드를 입력하다 잠시 멈췄을 때 AI가 뒤이어 올 코드를 흐릿하게 추천합니다. 제안이 마음에 들면 Tab 키로 바로 입력합니다.
